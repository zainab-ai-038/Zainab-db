# Car Rental Management CRUD Project (PHP + MySQL)

This is my assignment project where I made a CRUD application for Car Rental Management using PHP and MySQL. The app can add, view, update and delete car records.

## What I used
- XAMPP (Apache + MySQL)
- PHP
- MySQL
- HTML/CSS

## Files in this project
- database.sql - creates the database and cars table
- db.php - connects to the database
- index.php - shows all cars (main page)
- create.php - form to add a new car
- edit.php - form to update a car record
- delete.php - deletes a car record

## How to run it

1. Install XAMPP and start Apache + MySQL.
2. Put this folder inside htdocs.
3. Open phpMyAdmin, go to SQL tab, paste the content of database.sql and run it. This will create the database and table.
4. Open browser and go to: localhost/carrental-crud/index.php
5. From here you can add, edit and delete car records.

## Notes
- This is a CRUD (Create, Read, Update, Delete) assignment.
- I picked the Cars table and built all CRUD operations on it as required.
